### 一、项目下载
```shell
git clone git clone git@github.com:xhmily/aurora-psp-backend-ui.git
```

### 二、安装cnpm

> aurora-psp-backend-ui目录下执行

```shell
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

### 三、安装依赖

> 在aurora-psp-backend-ui目录下执行

```shell
cnpm install
```

### 四、启动

> 在pce-ui目录下执行

```shell
npm run dev
```

### 五、访问前端

```shell
http://localhost:8080
```
